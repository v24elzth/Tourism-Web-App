# Web Development Project - Fullstack - Platform-Based Application Development 

## 📝 Overview
This is a comprehensive web project covering both **Frontend** (UI/UX) and **Backend** (API/Server) development for a tourism/web application.

## 📂 Project Structure
*   **/backend**: Python-based backend handling routing and database operations.
*   **/frontend**: Client-side files including HTML structure, CSS styling, and JavaScript interactivity.
*   **/public**: A collection of images/assets used in the web interface.

## 🚀 How to Run
1. **Backend:**
   * Navigate to the `/backend` folder.
   * Run the application: `python app.py`
2. **Frontend:**
   * Open the `/frontend/html` folder.
   * Open the desired `.html` file in your web browser.

## 🛠 Tech Stack
*   **Backend:** Python (Flask/General Python)
*   **Frontend:** HTML, CSS, JavaScript
